# Workflow 3: Final Review

Trước khi hoàn tất task:

1. **Tự kiểm tra (Self-Check):**
   - Output có đúng yêu cầu không?
   - Có breaking changes hay side-effects nào không?
   - Architecture & naming conventions có được tuân thủ không?
   - Có security risk hoặc data leakage không?
2. **Đánh giá & Báo cáo (Evaluation & Report):**
   - Liệt kê các file đã thay đổi.
   - Trình bày những vấn đề kỹ thuật đã giải quyết.
   - Báo cáo rõ ràng các rủi ro tồn đọng (nếu có).
3. **Xin ý kiến hoàn thành:** Đợi Human review code và feedback bước cuối.
