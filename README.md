# News-Website-
Trang web tin tức trực tuyến được xây dựng bằng HTML, CSS và JavaScript thuần.
Đây là đồ án môn học với mục tiêu mô phỏng một hệ thống đọc tin tức cơ bản.

## ✨ Tính năng

- 📂 Đọc tin tức theo danh mục (thể thao, công nghệ, giải trí,...)
- 🔍 Tìm kiếm bài viết theo từ khóa
- 🛠️ Trang admin quản lý bài viết

## 🛠️ Công nghệ sử dụng

- HTML5, CSS3, JavaScript
- Font Awesome (icons)
- Google Fonts
